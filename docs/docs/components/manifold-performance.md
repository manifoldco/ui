# Performance

Collects performance related data of the components. Should sit just below
`<manifold-connection>` in the DOM.

e.g.
```html
<manifold-connection>
  <manifold-performance>
    <manifold-product product-label="logdna"></manifold-product>
  </manifold-performance>
</manifold-connection>
```
